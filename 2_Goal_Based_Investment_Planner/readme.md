# Goal Based Investment Planner

## Business Goal
1. Helps users plan investment based on life goals instead of guesswork.
2. Calculates required SIP for each financial goal.
3. factors inflation into long term planning.
4. Identifies funding shortfalls early.
5. Simulates multiple investment scenarios.
6. Prioritizes goals based on affordibility.
7. Provides personalized investment strategies.
8. Eables periodic progress tracking.
9. Reduces financial planning errors.
10. Makes financial planning understandable for non-experts.
