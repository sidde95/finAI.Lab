# Indian Market Intelligence Dashboard

## Business Case
1. Provides alive overview of the Infdian market.
2. Consolidates market data into one dashboard.
3. Displays sector performance in real time.
4. Tracks indices, gainer and losers.
5. Helps investors understand market trends quickly.
6. Provides market breadth indicators.
7. Integrates multiple market data sources.
8. Acts as the central information hub for later AI systems.
9. Enables better market monitoring.
10. Supports faster investment decisions.
---

## Technology Stack

| Category | Technologies |
|---|---|
| Frontend | React, TypeScript, Vite |
| UI Components | Tailwind CSS, shadcn/ui |
| Charts | Recharts, TradingView Lightweight Charts |
| Backend | Python, FastAPI |
| Indian Market Data | Upstox API |
| Historical Market Data | yfinance |
| Data Processing | Pandas, Polars |
| Database | PostgreSQL |
| Time-Series Database | TimescaleDB |
| Cache | Redis |
| Live Updates | FastAPI WebSockets |
| Data Visualization | Plotly |
| Metrics | Prometheus |
| Monitoring Dashboards | Grafana |
| Error Tracking | Sentry |
| Testing | Pytest, HTTPX, Playwright |
| CI/CD | GitHub Actions |
